Features

Reviews or Mentions Collection

The latest data is collected from Amazon, Rotten Tomatoes and other websites that appear as google search results. It collects reviews from Amazon, Rotten Tomatoes & the phrases that contain keyword in the websites that appear as google search results.

Data Preprocessing

The next step after data collection is to prepare the data so that it can feed to algorithms. The data is prepared in efficient way and only the useful data is fed to the algorithm to improve overall performance.

Fine Grained Sentiment Analysis

Fine Grained Sentiment Analysis algorithms detect sentiment at sentence & phrase level instead of document level. This way we can get better insights and more refined results. The sentences are categorized into five categories positive, partially positive, neutral, partially negative & negative.

Visualization

The most widely used and the efficient way of analyzing and intrepreting the data or results is to visualize. That way we can get better insights in no time & also useful for novice team members. The system provides various types of graphs to better understand your product and track feedback & generate visualization reports.
